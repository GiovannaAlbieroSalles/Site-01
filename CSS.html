  • Seletores Básicos

    Seletor por tag escolhe todos os elementos que correspondem ao nome fornecido.
    Sintaxe: tagname
    Seletor por classe escolhe elementos baseados no valor de seu atributo classe.
    Sintaxe: .classname
    Seletor por ID escolhe nós baseados no valor do atributo id. Deve existir apenas um elemento com o mesmo ID no mesmo documento.
    Sintaxe: #idname
    Seletores universais irá escolher todos os nós. Ele tamém existe em um namespace único e em uma variante de todo o namespace também.
    Sintaxe: * ns|* *|*
    Seletores por atributo ira escolher nós baseados no valor de um de seus atributos, ou até mesmo pelo próprio atributo.
    Sintaxe: [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]

  • Combinadores

    Seletores de irmãos adjacentes
    O combinador '+' seleciona os nós que seguem imediatamente o elemento especificado anteriormente.
    Sintaxe: A + B
    Seletores gerais de irmãos
    O combinador '~' seleciona os nós que seguem (não necessariamente imediatamente) o elemento e specificado anteriormente, se ambos os elementos compartilham o mesmo pai.
    Sintaxe: A ~ B
    Seletor de filhos
    O combinador '>' selecina nós que são filhos diretos do elemento especificado anteriormente.
    Sintaxe: A > B
    Seletor de descendentes
    O combinador ' ' seleciona os nós que são filhos do elemento especificado anteriormente (não é necessário que seja um filho direto).
    Sintaxe: A B

  • Pseudo-elementos

    Pseudo-elementos são asbtrações da árvore que representa entidades além do que o HTML faz. Por exemplo, o HTML não tem um elemento que descreve a primeira letra ou linha de um parágrafo, ou o marcados de um lista. Os pseudo-elementos representam essas entidades e permitem que as regras CSS sejam associadas a elas. Desta forma, essas entidades podem ser denominadas independentemente.

  • Pseudo-classes

    Pseudo-classes permitem selecionar elementos baseados em indormações que não estão contidas na árvore de documentos como um estado ou que é particularmente complexa de extrair. Por exemplo, eles correspondem se um link foi visitado anteriormente ou não.

  • Posicionamento

    Permite que você retire elementos do fluxo normal do layout do documento e faça com que eles se comportem de forma diferente, por exemplo, sentados um sobre o outro ou sempre permanecendo no mesmo local dentro da janela de exibição do navegador. Este artigo explica os diferentes valores de posição e como usá-los.

  • Relativo

    Uma vez que o elemento posicionado tomou seu lugar no fluxo de layout normal, você pode então modificar sua posição final, incluindo tornando-se sobreposição de outros elementos na página.

  • Absoluto

    Fixa um elemento no lugar em relação ao elemento <html> ou seu ancestral mais próximo posicionado

  • Estático

    Posicionamento estático é o padrão que cada elemento recebe - apenas significa "colocar o elemento em sua posição normal no fluxo de layout do documento - nada de especial para ver aqui".

  • Fixo

    Corrige um elemento no lugar em relação à própria janela de visualização do navegador. Isso significa que você pode criar itens de interface do usuário úteis que são fixos no local, como menus de navegação persistentes.

  • Box

    Cada elemento dentro de um documento é estruturado como uma box retangular dentro do layout do documento, o tamanho e "camadas de cebola" podem ser tweaked usando algumas propriedades CSS específicas.

  • Padding

    Representa a margem interna de uma caixa CSS - a camada entre a borda externa da caixa de conteúdo ea borda interna da borda. O tamanho desta camada pode ser definido em todos os quatro lados ao mesmo tempo com a propriedade de abreviação de preenchimento, ou um lado de cada vez com as propriedades padding-top, padding-right, padding-bottom e padding-left.

  • Border

    É uma camada distinta, sentada entre a borda externa do preenchimento e a borda interna da margem. Por padrão, a borda tem um tamanho de 0 - tornando-o invisível - mas você pode definir a espessura, estilo e cor da borda para torná-lo aparecer. A propriedade de abreviação de borda permite que você defina todas estas em todos os quatro lados ao mesmo tempo, por exemplo borda: 1px preto sólido. Isso pode ser dividido em inúmeras propriedades de largura diferentes para necessidades de estilo mais específicas.

  • Margin

    Representa a área externa que rodeia a caixa CSS, que empurra para cima contra outras caixas CSS no layout. Ele se comporta como um preenchimento; A propriedade abreviada é margem e as propriedades individuais são margin-top, margin-right, margin-bottom e margin-left.
